## 🚦 Getting Started

### Prerequisites

- 📌 CMake (3.10 or higher)
- 📌 C++20 compatible compiler
- 📌 Git

### Quick Start

1️⃣ Clone the repository:
```bash
git clone https://github.com/yourusername/modern-cpp-path.git
cd modern-cpp-path
```

2️⃣ Build the project:
```bash
cmake -B build -S .
cmake --build build
```
